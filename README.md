# PHP Dasar Journey

Repository ini berisi perjalanan saya belajar PHP dari nol.

## Progress
- Day 1 : Tipe Data & Variable
- Day 2 : Condition
- Day 3 : Looping
- Day 4 : Array
- Day 7 : Function

## Goals
Konsisten belajar tiap hari walaupun hanya 30 menit untuk membentuk habit dan mencapai tujuan menjadi programmer.

## How to run
1. Clone repositori ini:
   git clone https://github.com/husnifatah-dev/php-journey
2. Masuk ke folder project:
   cd nama-folder
3. Jalankan server PHP:
   php -S localhost:8000
4. Buka di browser:
   http://localhost:8000
5. Untuk membuka file tertentu:
   http://localhost/nama_file.php 

## Note
Semua code ditulis manual untuk memahami konsep, bukan hanya copy-paste, serta melatih muscle memory.
